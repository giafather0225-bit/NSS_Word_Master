LEARNLN_01concrete
What Is a Pattern in a Table?
A pattern in a table shows a rule that connects the input and output. For example, if each pan holds 6 muffins, the rule is 'Multiply by 6'. For every additional pan (input), you get 6 more muffins (output).
🚩 mark
LEARNLN_02concrete
Finding the Rule
Look at a known pair: input 2 → output 14. Ask: 2 times what equals 14? 2 × 7 = 14. Check with another pair: input 4 → output 28, and 4 × 7 = 28. The rule is 'Multiply by 7'.
🚩 mark
LEARNLN_03pictorial
Completing a Table
Once you know the rule, use it to fill in blanks. If the rule is × 7 and the input is 5, then 5 × 7 = 35. Always verify: does the same rule work for every row?
🚩 mark
LEARNLN_04abstract
Patterns Go Both Ways
In a multiplication table, as the input goes up by 1, the output goes up by the rule number. Rule ×8: input 3 → 24, input 4 → 32 (up by 8). If the input doubles, the output doubles too: input 2 → 16, input 4 → 32.
🚩 mark
LEARNLN_05abstract
Using Patterns to Predict
Tables let you predict unknown values instantly. If rule is ×7 and you see input 5 → output 35, you know input 6 → output 42 (just add 7). To find a missing input, reverse the rule: output ÷ rule = input. 63 ÷ 7 = 9.

LEARNLN_01concrete
What Is an Unknown Factor?
In the equation n × 3 = 12, the unknown factor is n. You need to find what number times 3 equals 12. The letter (n, m, x, etc.) is a placeholder for the missing factor.
🚩 mark
LEARNLN_02concrete
Strategy: Skip Count to Find It
To find n in n × 5 = 30, skip count by 5s: 5, 10, 15, 20, 25, 30 — that is 6 jumps. So n = 6. Or use a known fact: 6 × 5 = 30.
🚩 mark
LEARNLN_03pictorial
Strategy: Use an Array
For 3 × □ = 24, draw 3 rows. Keep adding columns until you reach 24 dots total — you will get 8 columns. So □ = 8. Arrays make the unknown factor visible.
🚩 mark
LEARNLN_04abstract
Division Is the Reverse
Finding an unknown factor is the same as dividing. If n × 6 = 42, then n = 42 ÷ 6 = 7. Division 'undoes' multiplication. You can always use this shortcut to solve quickly.
🚩 mark
LEARNLN_05abstract
Always Check Your Answer
After finding the unknown factor, substitute it back: if n = 7, check n × 6 = 7 × 6 = 42 ✓. If it matches the product, your answer is correct. This works for any unknown factor equation.

The Distributive Property
You can break apart one factor to make multiplication easier. Example: 6 × 30 = 6 × (10 + 10 + 10) = 60 + 60 + 60 = 180. Breaking the factor into smaller parts gives the same product.

Using Place Value
Think of 30 as 3 tens. So 6 × 30 = 6 × 3 tens = 18 tens = 180. The shortcut: multiply the one-digit factor by the non-zero digit, then attach a zero. 6 × 3 = 18 → 6 × 30 = 180.

Solving Word Problems

LEARNLN_04abstract
Break Into Any Two Parts
You can split a factor into any two addends: 6 × 7 = 6 × (3 + 4) = 18 + 24 = 42. Or split a multiple of 10: 7 × 30 = 7 × (20 + 10) = 140 + 70 = 210. Any split works as long as the parts add up to the original factor.

Why It Always Works
The rule is: a × (b + c) = (a × b) + (a × c). Example: 5 × (20 + 30) = (5 × 20) + (5 × 30) = 100 + 150 = 250. This is the same as 5 × 50 = 250. The Distributive Property always gives the same answer.
