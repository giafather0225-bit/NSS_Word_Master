LEARNLN_01concrete난이도 0
What Are Equal Groups?
Equal groups all have the SAME number of objects. 3 plates with 4 cookies each = 3 equal groups of 4. If one plate has 5 cookies and the others have 4, the groups are NOT equal.
🚩 mark
LEARNLN_02concrete난이도 0
Skip Counting to Find the Total
Skip count by the number in each group. For 4 groups of 5: count 5, 10, 15, 20. That's the total — 20 in all.
🚩 mark
LEARNLN_03pictorial난이도 0
Groups Of
'3 groups of 6' means 3 groups, each with 6 objects. Total = 6 + 6 + 6 = 18. This is the same as 3 × 6 = 18.
🚩 mark
LEARNLN_04pictorial난이도 0
Repeated Addition
Equal groups are the same as adding the same number over and over. 5 groups of 3 = 3 + 3 + 3 + 3 + 3 = 15. This idea is the start of multiplication.
🚩 mark
LEARNLN_05abstract난이도 0
Reading Problems Carefully
Look for words like 'each', 'every', 'per' — they tell you equal groups. 'Each box has 4 pencils, 5 boxes' → 5 groups of 4, not 4 groups of 5. The number of GROUPS is 5; the SIZE of each group is 4.

Skip Counting on a Number Line
A number line starts at 0. Each jump is the same size. To find 3 × 4, make 3 jumps of 4: 0 → 4 → 8 → 12. You land on 12, so 3 × 4 = 12. The number you land on is always the product.

Bar Models for Multiplication
A bar model is a rectangle divided into equal sections. Each section = the size of one group. The number of sections = the number of groups. Add all sections to find the total (the product).

Two-Step Problems: Multiply Then Add or Subtract
Some problems need two steps. Example: 3 bags of 5 apples, eat 2. Step 1: 3 × 5 = 15 apples. Step 2: 15 − 2 = 13 left. Always multiply first to find the group total, then do the second operation.

Choosing the Right Model
For equal groups → circles and dots. For side-by-side groups → arrays. For 'how many total in parts' → bar models. All three models show the same multiplication, just drawn differently. Pick the one that fits the story.

What is an Array?
An array is a way to show objects in equal rows and columns. Rows go across (left to right). Columns go up and down. Every row has the same number of objects. Example: 3 rows of 4 dots = 12 dots total.
🚩 mark
LEARNLN_02concrete난이도 0
Reading an Array
To read an array: (1) Count the rows — how many rows go across? (2) Count the columns — how many are in each row? (3) Write the multiplication: rows × columns = product. Example: 4 rows of 3 → 4 × 3 = 12.
🚩 mark
LEARNLN_03pictorial난이도 0
Arrays and Multiplication Sentences
An array with 5 rows and 2 columns shows 5 × 2 = 10. You can also read it as 2 × 5 = 10 (columns × rows). The product is the same either way! Both multiplication sentences describe the same array.
🚩 mark
LEARNLN_04pictorial난이도 0
Rotating an Array
Rotate a 3×4 array 90° and it becomes a 4×3 array — the product stays 12! This shows the Commutative Property: 3 × 4 = 4 × 3. Arrays prove it visually: same dots, just rearranged in rows and columns.
🚩 mark
LEARNLN_05abstract난이도 0
Arrays in Word Problems
Word problems hide arrays. Look for clue words: 'rows' tells you the first factor; 'in each row' or 'columns' tells you the second factor. Example: '5 rows of seats with 4 seats in each row' → draw a 5×4 array → 5 × 4 = 20 seats.

The Commutative Property
When you multiply two numbers, you can change their order and the product stays the same. Example: 3 × 5 = 15 and 5 × 3 = 15. This is the Commutative Property of Multiplication. The word 'commutative' comes from 'commute' — like moving back and forth.
🚩 mark
LEARNLN_02concrete난이도 0
Why It Works — Arrays
A 3×5 array has 3 rows of 5. Rotate it 90° and you get a 5×3 array (5 rows of 3). It's the same dots — just rearranged! That is why the product never changes when you swap factors.
🚩 mark
LEARNLN_03pictorial난이도 0
Turn-Around Facts
Every multiplication fact has a 'turn-around' partner. If you know 7 × 4 = 28, you also know 4 × 7 = 28. Learning one fact gives you a second fact for free! This cuts the number of facts you need to memorize almost in half.
🚩 mark
LEARNLN_04abstract난이도 0
Finding the Missing Factor
The Commutative Property helps you fill in blanks. Example: 9 × ? = ? × 9 = 63. Since 9 × 7 = 63, the missing factor is 7. Both sides of the equation show the same product: 9 × 7 = 7 × 9 = 63.
🚩 mark
LEARNLN_05abstract난이도 0
When Does It NOT Work?
The Commutative Property works for multiplication (and addition), but NOT for subtraction or division. Example: 8 − 3 = 5, but 3 − 8 ≠ 5. And 12 ÷ 3 = 4, but 3 ÷ 12 ≠ 4. Always check which operation you are using!
