What is a Tally Table?
A tally table uses tally marks to record data. Each mark | represents 1. A group of five is written as 𝍸 (four marks with a line through them). To find the total, count the tally marks.
🚩 mark
LEARNLN_02concrete난이도 0
What is a Frequency Table?
A frequency table uses numbers to show how many times each item was chosen. It has two columns: the category and the number (frequency). You can convert a tally table into a frequency table by counting the tally marks.

Comparing Data
To find 'how many more' or 'how many fewer,' subtract the smaller number from the larger number. To find a total, add all the numbers together.

What is a Bar Graph?
A bar graph uses bars to show data. The length or height of each bar shows the value. One axis shows categories, the other shows numbers (the scale).
🚩 mark
LEARNLN_02concrete난이도 0
Reading the Scale
The scale is the set of numbers along one axis. It can count by 1s, 2s, 5s, 10s, etc. To read a bar's value, look at where the bar ends and match it to the scale.
🚩 mark
LEARNLN_03pictorial난이도 0
Comparing Bars
The tallest (or longest) bar represents the most. The shortest bar represents the least. To find 'how many more' or 'how many fewer,' subtract the two values.

Steps to Make a Bar Graph
Step 1: Write the title. Step 2: Label the axes (categories on one axis, numbers on the other). Step 3: Choose a scale that fits the data. Step 4: Draw the bars to the correct height. Step 5: Check that each bar matches the data.
🚩 mark
LEARNLN_02concrete난이도 0
Choosing a Scale
Look at the largest number in your data. Pick a scale that goes up to or just past that number. If the largest number is 12, counting by 2s (0–12) is good. If the largest is 30, counting by 5s (0–30) works well.
🚩 mark
LEARNLN_03pictorial난이도 0
Drawing Accurate Bars
Each bar must end at the exact number on the scale. If the scale is by 2s and you need to show 5, draw the bar halfway between 4 and 6. All bars should be the same width.
