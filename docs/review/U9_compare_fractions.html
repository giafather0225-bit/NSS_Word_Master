LEARNc9_l2_learn_01
When two fractions have the SAME DENOMINATOR, the pieces are the same size. You just compare how many pieces each fraction has — that's the numerator. More pieces = greater fraction. Example: 2/8 vs 6/8 → same size pieces, but 6 pieces > 2 pieces, so 6/8 > 2/8.
🚩 mark
LEARNc9_l2_learn_02
You can use a fraction strip to see this. Imagine a bar split into 4 equal parts. Color 1 part → 1/4. Color 3 parts → 3/4. The bar with 3 parts colored is clearly more filled. So 3/4 > 1/4.
🚩 mark
LEARNc9_l2_learn_03
If both fractions have the same numerator AND the same denominator, they are equal. Example: 3/4 = 3/4. This also works in word problems: if two people each eat 2/6 of the same cake, they ate the same amount.
