# Academy — 미국학교 대비 영어 학습 스펙

> 목적: 지아(9세, 영어 모국어, 한국 외국인학교)의 미국 초등 3~4학년 적응을 위한 Academic English 학습
> DUX Words(학원용)와 완전히 별개 섹션. 필요한 컴포넌트(SM-2, TTS, XP 등)는 재사용 가능.

## 2. 학습 목표 (Learning Goals)

미국 Common Core 3~4학년 기준:
- Tier 2 Academic Vocabulary 300개 완전 습득
- Fiction / Nonfiction 독해 (200~400 words)
- 텍스트 근거 기반 짧은 쓰기 응답
- 어형 변화(Morphology) 패턴 인식
- 간격 반복으로 장기 기억 정착

---

## 3. 학습 3축 (Three Axes)

### 축 1 — Academic Vocabulary
### 축 2 — Reading + Writing
### 축 3 — Word Depth

진행 순서: 축 1 → 축 2 → 축 3 (레벨 완료 후 해금)

## 5. 단어 학습 흐름 (축 1)

### 5-Step Word Learning

```
① Meet It   — 단어 + 영영 정의(쉬운 말) + 이미지/아이콘 + TTS 발음
② See It    — 교과 예문 2개 읽기 (서로 다른 과목 맥락)
③ Use It    — 예문 빈칸 채우기
④ Know It   — 동의어/반의어 4지선다
⑤ Own It    — 올바른 사용 예시 2지 선택
```

**+ Morphology 힌트** (매 단어 하단 고정)
- 접두사/접미사/어근 1줄 표시
- 예: `predict` → `pre- (before) + dict (say)` → prediction, predictable

### 진도 구조

```
단어 5개 학습
    ↓
Mini Quiz (5문제, 혼합형)
    ↓
단어 5개 학습
    ↓
Mini Quiz
    ↓
단어 5개 학습
    ↓
Unit Test (15단어 종합)
    ↓
Reading Session (축 2 해금) ← 배운 단어 포함 지문
    ↓
다음 15단어
```

### Level 진행

```
Level 1 (95개) → 7 Unit Tests → Level 1 Final Test
Level 2 (85개) → 6 Unit Tests → Level 2 Final Test
Level 3 (120개) → 8 Unit Tests → Level 3 Final Test
```

## 6. 독해 + 쓰기 흐름 (축 2)

Unit Test 통과 후 Reading Session 해금.

```
지문 읽기 (200~400 words, CLEAR Corpus 기반)
    ↓
이해 질문 3개
  - Q1: Main idea (객관식)
  - Q2: Key detail (객관식)
  - Q3: Inference (단문 쓰기 1~2문장)
    ↓
Mini Write
  "Using evidence from the text, write 1-2 sentences to answer Q3."
    ↓
다음 Unit으로
```

### 지문 선정 기준 (CLEAR Corpus 필터)
- Lexile: 520L ~ 820L (3~4학년 범위)
- 길이: 200~400 words
- 장르: Fiction / Nonfiction 번갈아
- 해당 Unit 단어 최소 2개 이상 포함

---

## 7. Word Depth (축 3)

Level Final Test 통과 후 해금.
해당 Level 핵심 단어 20개 선별, 심화 학습.

```
각 단어당:
- 어원 (etymology) 상세
- 콜로케이션 3개 (자주 함께 쓰이는 표현)
- 뉘앙스 차이 (동의어 간 차이점)
- 실제 교과서 문장 예시
```

---

## 8. 간격 반복 (Spaced Repetition)

DUX Words의 SM-2 엔진 재사용, Academy에도 적용.

- Unit Test / Mini Quiz 오답 → SM-2 복습 큐 자동 등록
- 매일 학습 시작 시 복습 큐 표시 (오늘의 복습 N개)
- 6주 재노출 사이클로 장기 기억 정착

## 11. 개발 우선순위

```
Phase 1 — 데이터 파이프라인
  - Merriam-Webster API 연동
  - WordNet 연동
  - CLEAR Corpus 필터링 (3~4학년 지문 추출)
  - 300단어 morphology 수동 큐레이션

Phase 2 — 축 1 (Academic Vocabulary)
  - 5-Step Word Learning UI
  - Mini Quiz
  - Unit Test
  - SM-2 연동

Phase 3 — 축 2 (Reading + Writing)
  - 지문 뷰어
  - 이해 질문 + Mini Write
  - 지문-단어 연결 시스템

Phase 4 — 축 3 (Word Depth)
  - Level Final Test
  - Word Depth 심화 뷰
```
